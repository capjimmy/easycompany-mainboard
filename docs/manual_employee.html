2. 로그인

  1. 프로그램을 실행합니다.
  2. 관리자로부터 받은 아이디와 비밀번호를 입력합니다.
  3. 로그인 버튼을 클릭합니다.
초기 비밀번호는 사번 또는 "1234"입니다. 로그인 후 설정에서 반드시 비밀번호를 변경해주세요.

3. 통합검색

메뉴: 홈 > 통합검색

  • 검색어를 입력하면 계약서, 견적서, 외주, 파일을 한번에 검색합니다.
  • 결과를 유형별로 필터링할 수 있습니다.
  • 검색 결과 클릭 시 해당 상세 페이지로 이동합니다.

4. 견적관리

메뉴: 계약/매출 > 견적관리

견적서 작성하기

  1. 견적서 작성 버튼을 클릭합니다.
  2. 수신처를 입력합니다 (거래처명 입력 시 자동완성).
  3. 용역명과 견적일자를 입력합니다.
  4. 인건비 항목 추가: 등급을 선택하면 단가가 자동 입력됩니다. 인원수, 참여율, 개월수를 입력합니다.
  5. 경비 항목 추가: 항목을 선택하고 금액을 입력합니다. 비율 항목은 인건비 기준으로 자동계산됩니다.
  6. 필요시 상세내역을 추가합니다 (대분류 > 세부 > 세세부).
  7. 하단에서 총금액을 확인하고 저장을 클릭합니다.

견적서 제출 및 이메일

이메일 발송 시 승인 요청이 필요합니다.
사원은 이메일을 직접 발송할 수 없으며, "승인 요청" 버튼을 클릭하면 부서관리자에게 승인 요청이 전송됩니다. 부서관리자가 승인하면 자동으로 이메일이 발송됩니다.

견적서 삭제, 승인/반려, 계약전환은 관리자만 가능합니다.

5. 계약관리

메뉴: 계약/매출 > 계약관리

  • 계약 목록 조회 및 검색
  • 새 계약 등록 가능
  • 계약 상세 정보 확인
  • 월별현황, 계약변경이력 조회
계약 수정, 상태변경, 입금등록, 삭제는 부서관리자 이상만 가능합니다.

6. 거래처관리

메뉴: 계약/매출 > 거래처관리

  • 거래처 목록 조회 및 검색
  • 새 거래처 추가 가능
거래처 수정/삭제는 관리자만 가능합니다.

7. 인사/총무

연차 신청

메뉴: 인사/총무 > 연차 신청

  1. 상단에서 잔여 연차일수를 확인합니다 (입사일 기준 자동 계산).
  2. 연차 신청 버튼을 클릭합니다.
  3. 휴가 유형을 선택합니다: 연차, 오전반차, 오후반차, 병가, 특별휴가
  4. 날짜를 선택하면 일수가 자동 계산됩니다 (반차는 0.5일).
  5. 사유를 입력하고 신청을 클릭합니다.
승인 절차: 신청 → 부서관리자 부서승인 → 회사관리자 최종승인
대기중 또는 부서승인 상태에서 본인이 취소할 수 있습니다.

메일 승인

메뉴: 인사/총무 > 메일 승인

"내 요청" 탭에서 본인이 요청한 이메일 발송 건의 승인 상태를 확인합니다.

증명서 발급

메뉴: 인사/총무 > 증명서 발급

  1. 증명서 신청 버튼 클릭
  2. 증명서 유형 선택: 재직증명서, 경력증명서
  3. 용도를 입력합니다.
  4. 신청 후 관리자 승인을 기다립니다.

8. 캘린더

계약 캘린더

계약 시작/종료일, 납부일 등의 일정을 달력에서 확인합니다.

인사 캘린더

팀원들의 연차/병가/출장 일정을 확인합니다.

공간 캘린더 (회의실 예약)

  1. 날짜와 회의실을 선택합니다.
  2. 빈 시간대를 클릭하여 예약합니다.
  3. 회의 제목, 참석 인원, 장비 필요 여부 등을 입력합니다.
  4. 본인이 예약한 건만 삭제할 수 있습니다.

9. 프로젝트

메뉴: 프로젝트 > 프로젝트 현황

  • 진행중/완료/타절 프로젝트 목록 조회
  • 프로젝트별 진행률, 수금 현황 확인
  • 하위작업(서브태스크) 현황 확인
프로젝트 상태 변경 및 수정은 부서관리자 이상만 가능합니다.

10. 설정

메뉴: 시스템 > 설정

프로필 수정

전화번호, 이메일을 수정할 수 있습니다.

비밀번호 변경

  1. 설정 > 보안 탭으로 이동합니다.
  2. 현재 비밀번호를 입력합니다.
  3. 새 비밀번호를 입력하고 확인합니다.
  4. 변경을 클릭합니다.
초기 비밀번호(사번 또는 1234)를 사용중이라면, 보안을 위해 반드시 변경해주세요.

테마 변경

라이트/다크 모드를 선택할 수 있습니다.
