1. 권한 개요

부서관리자는 소속 부서의 업무를 관리하고, 연차 신청에 대한 1차(부서) 승인 권한을 가집니다.

기능권한비고
견적서/계약서조회/생성/수정삭제 불가
거래처조회/생성/수정삭제 불가
연차 승인부서 승인 (1단계)최종승인은 회사관리자
메일 승인승인/반려같은 부서원 요청만
프로젝트조회/수정상태변경 가능
부서관리조회만수정 불가
문서 템플릿업로드/관리
대시보드접근 불가관리자 전용
AI검색접근 불가관리자 전용
사용자/회사/권한관리접근 불가관리자 전용

2. 홈 메뉴

  • 프로젝트 현황: 계약 목록 기반 프로젝트 현황 확인
  • 통합검색: 계약/견적/외주/파일 통합 검색
대시보드, AI검색은 관리자 전용 메뉴입니다.

3. 견적관리

가능한 작업

  • 견적서 작성 (인건비 + 경비 + 상세내역)
  • 견적서 수정 (초안 상태)
  • 견적서 제출
  • 이메일 발송 / PDF 미리보기
  • 견적서 복제
견적서 삭제 및 승인/반려는 회사관리자 이상만 가능합니다.

4. 계약관리

  • 계약 등록 및 수정
  • 진행상태 변경
  • 입금 등록
  • 외주 등록/관리
  • 월별현황 조회 / 변경이력 조회
계약 삭제는 회사관리자 이상만 가능합니다.

5. 거래처관리

  • 거래처 조회 및 추가
  • 거래처 정보 수정
거래처 삭제는 회사관리자 이상만 가능합니다.

6. 인사/총무

연차 신청

본인 연차 신청 가능. 잔여일수 자동 계산됩니다.

연차 승인관리 (부서 승인)

부서관리자의 승인 역할:
  • 같은 부서 직원의 연차 신청에 대해 부서 승인(1단계) 처리
  • 대기중(pending) → 부서승인(dept_approved) 상태로 변경
  • 반려 시 사유를 입력하면 신청자에게 알림 전송
  • 부서 승인 후 회사관리자의 최종승인이 필요합니다

메일 승인

같은 부서 직원의 이메일 발송 요청을 승인/반려합니다. 승인 시 자동 발송됩니다.

본인 증명서 신청 가능 (승인은 회사관리자).

7. 캘린더

  • 계약 캘린더: 계약 일정 확인
  • 인사 캘린더: 소속 부서의 연차/병가 일정 확인
  • 공간 캘린더: 회의실 예약 및 관리

8. 프로젝트

  • 프로젝트 현황 확인 (진행률, 수금현황)
  • 프로젝트 상태 변경 가능
  • 하위작업 확인
  • 일정관리 타임라인

9. 관리

  • 부서관리: 소속 부서 정보 조회 (수정 불가)
사용자관리, 권한설정, 회사관리는 접근할 수 없습니다.

10. 설정

  • 문서 템플릿: 부서용 문서 템플릿 업로드 및 관리
  • 일반 설정: 프로필, 비밀번호 변경, 테마 설정
양식설정, 백업관리는 회사관리자 이상만 접근 가능합니다.
