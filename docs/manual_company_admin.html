기능권한

회사관리자는 소속 회사의 데이터만 조회/관리할 수 있습니다. 다른 회사의 데이터에는 접근할 수 없습니다.

2. 대시보드

메뉴: 홈 > 대시보드

  • 소속 회사의 계약/견적 현황, 수금율, 월별 통계 확인
  • 기간 필터: 이번 달 / 이번 분기 / 올해 / 작년 / 직접 선택
  • 최근 프로젝트 및 견적 현황 카드

3. 견적관리

메뉴: 계약/매출 > 견적관리

가능한 작업

작업설명
견적서 작성기본정보 + 인건비 + 경비 + 상세내역 입력
수정/삭제초안 상태 견적서 수정 및 삭제
제출/승인/반려견적서 상태 관리
계약 전환승인된 견적서를 계약서로 전환
이메일 발송PDF 첨부하여 이메일 직접 발송
PDF 미리보기견적서 미리보기 및 다운로드
복제기존 견적서를 복사하여 새로 작성

견적서 작성 순서

  1. 견적서 작성 버튼 클릭
  2. 수신처 입력 (거래처 자동완성 지원)
  3. 용역명, 견적일자, 유효기간 입력
  4. 인건비 항목 추가 (등급 선택 → 자동 단가 적용)
  5. 경비 항목 추가 (수동입력 또는 비율자동계산)
  6. 상세내역 추가 (필요시 대분류/세부/세세부)
  7. 하단의 총금액 확인 후 저장

4. 계약관리

메뉴: 계약/매출 > 계약관리

  • 계약 등록/수정/삭제
  • 진행상태 변경 (준비중/진행중/완료/타절)
  • 입금 등록 및 대금조건 관리
  • 계약서 PDF 생성 및 이메일 발송
  • 하위작업(서브태스크) 관리

관련 하위메뉴

  • 월별현황: 월별 계약/수금 통계
  • 계약변경이력: 계약 변경 내역 타임라인
  • 외주관리: 외주업체 등록/비용 관리

5. 거래처관리

메뉴: 계약/매출 > 거래처관리

  • 거래처 추가: 회사명, 유형, 사업자번호, 담당자 정보
  • 거래처별 계약금액/수금현황 자동 집계
  • 삭제 가능 (회사관리자 권한)

6. 인사/총무

연차 신청

본인 연차 신청 및 잔여일수 확인

연차 승인관리

회사관리자의 승인 권한:
  • 대기중(pending) → 직접 최종승인 가능
  • 부서승인(dept_approved) → 최종승인
  • 모든 상태에서 반려 가능 (반려 사유 입력)

메일 승인

직원이 요청한 이메일 발송 건 승인/반려. 승인 시 자동 발송됩니다.

증명서 발급

직원의 재직/경력 증명서 신청 건 승인/반려

7. 캘린더

  • 계약 캘린더: 계약 일정, 납부일, 마일스톤
  • 인사 캘린더: 전 직원 연차/병가 일정 (소속 회사)
  • 공간 캘린더: 회의실 예약 현황 및 예약

8. 프로젝트

  • 프로젝트 현황: 진행중/완료/타절 탭, 진행률/수금 현황
  • 프로젝트 상태 변경 가능
  • 일정관리: 프로젝트 타임라인

9. 관리

사용자관리

  • 소속 회사 사용자 추가/수정/삭제
  • 역할 설정: 사원, 부서관리자 (회사관리자 역할 부여는 슈퍼관리자만)
  • 직원명부 엑셀 일괄 가져오기
  • 비밀번호 재설정

부서관리

  • 부서 추가/수정/삭제
  • 부서장 지정

권한설정

  • 역할별 메뉴 접근 권한 설정
회사관리 메뉴는 슈퍼관리자 전용이므로 접근할 수 없습니다.
