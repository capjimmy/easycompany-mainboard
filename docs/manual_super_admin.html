1. 시스템 개요

건설경제연구원 ERP는 계약/견적 관리, 인사관리, 프로젝트 관리를 통합한 사내 업무 시스템입니다.

슈퍼관리자 권한 범위

영역권한
모든 메뉴전체 접근 가능
모든 데이터회사 구분 없이 전체 조회/수정/삭제
사용자 관리모든 회사의 사용자 생성/수정/삭제
회사 관리회사 추가/수정/삭제 (슈퍼관리자 전용)
시스템 설정SMTP, API키, 백업 등 전체 설정

2. 로그인

  1. 프로그램을 실행합니다.
  2. 아이디와 비밀번호를 입력합니다.
  3. 로그인 버튼을 클릭합니다.
비밀번호를 잊은 경우, 슈퍼관리자는 관리 > 사용자관리에서 직접 비밀번호를 재설정할 수 있습니다.

3. 대시보드

메뉴: 홈 > 대시보드

표시 정보

  • 계약 현황 (진행중/완료/타절 건수)
  • 총 계약금액 및 수금율
  • 견적 현황
  • 월별 계약/수금/미수금 요약
  • 최근 진행중 프로젝트

기간 필터

상단의 이번 달 이번 분기 올해 작년 버튼 또는 날짜 범위 선택기를 사용합니다.

4. 견적관리

메뉴: 계약/매출 > 견적관리

견적 목록

  • 검색: 견적번호, 수신처, 용역명으로 검색
  • 필터: 상태별 (초안/제출/협상중/승인/반려/전환됨)
  • 날짜 범위 필터

견적서 작성

  1. 견적서 작성 버튼 클릭
  2. 기본정보 입력: 수신처 (자동완성), 용역명, 견적일자
  3. 인건비 항목 추가: 등급 선택 → 인원/참여율/개월 입력 → 소계 자동계산
  4. 경비 항목 추가: 항목 선택 → 금액 입력 (비율계산은 자동)
  5. 상세내역 추가 (선택): 대분류 > 세부 > 세세부 계층 구조
  6. 저장 클릭 → 총금액(인건비+경비+상세내역+VAT) 자동 계산

견적서 액션

액션설명
제출초안 → 제출 상태로 변경
승인/반려제출된 견적서 승인 또는 반려
계약전환승인된 견적서를 계약서로 전환
이메일 발송견적서를 PDF 첨부하여 이메일 발송
복제기존 견적서를 복사하여 새로 작성
PDF 미리보기견적서 PDF 미리보기 및 다운로드
삭제견적서 삭제 (관리자 전용)

5. 계약관리

메뉴: 계약/매출 > 계약관리

계약 등록

  1. 계약 등록 버튼 클릭
  2. 기본정보: 계약유형, 발주기관, 용역명, 기간
  3. 금액: 계약금액 → VAT/총액 자동계산
  4. 대금조건: 착수금/중도금/잔금 등록
  5. 문서: PDF 생성, 첨부파일 관리
  6. 하위작업(서브태스크) 등록

계약 상태 관리

상태설명
준비중계약 준비 단계
진행중계약 수행 중
완료계약 완료
타절계약 취소/중단

하위메뉴

메뉴기능
월별현황월별 계약/수금/미수금 집계표
계약변경이력모든 계약의 변경 내역 타임라인
외주관리외주업체 등록/관리, 외주비 관리

6. 거래처관리

메뉴: 계약/매출 > 거래처관리

  • 거래처 추가: 회사명, 유형(매출처/매입처/겸용), 사업자번호, 담당자 등
  • 거래처별 계약금액/수금현황 자동 집계
  • 거래처 상세에서 관련 계약 목록 확인
  • 삭제: 슈퍼관리자/회사관리자만 가능

7. 인사/총무

연차 신청 (홈 > 인사/총무 > 연차 신청)

  • 본인 연차 신청 및 잔여 연차 확인
  • 신청 유형: 연차, 오전반차, 오후반차, 병가, 특별휴가

연차 승인관리 (인사/총무 > 연차 승인관리)

슈퍼관리자는 모든 회사의 연차를 최종 승인/반려할 수 있습니다.

2단계 승인 체계: 부서관리자(부서승인) → 회사관리자/슈퍼관리자(최종승인). 슈퍼관리자는 대기중 상태에서 바로 최종승인도 가능합니다.

메일 승인 (인사/총무 > 메일 승인)

  • 직원이 요청한 이메일 발송 건 승인/반려
  • 승인 시 자동 발송

증명서 발급 (인사/총무 > 증명서 발급)

  • 직원 증명서 신청 승인/반려
  • 증명서 유형: 재직증명서, 경력증명서

8. 캘린더

캘린더표시 내용
계약 캘린더계약 시작/종료일, 납부일, 마일스톤
인사 캘린더직원 연차/병가/출장 일정
공간 캘린더회의실 예약 현황 및 예약 관리

9. 프로젝트

메뉴: 프로젝트 > 프로젝트 현황

  • 진행중/완료/타절 탭별 프로젝트 카드 목록
  • 프로젝트별 진행률, 수금현황, 하위작업 상태 확인
  • 프로젝트 상태 변경 (진행중 → 완료/타절 등)
  • 일정관리: 프로젝트 타임라인 뷰

10. 관리 (슈퍼관리자 전용)

사용자관리

  1. 사용자 추가 버튼 클릭 또는 직원명부 가져오기로 일괄 등록
  2. 필수 입력: 아이디, 비밀번호, 이름, 소속부서, 역할
  3. 역할 설정: 사원 / 부서관리자 / 회사관리자
  4. 입사일, 퇴사일, 직급 등 부가정보 입력
직원명부 가져오기 시 초기 비밀번호는 사번 또는 "1234"로 설정됩니다.

부서관리

  • 부서 추가/수정/삭제
  • 상위부서 설정 (계층 구조)
  • 부서장 지정

권한설정

  • 메뉴별 역할 권한 조회/수정 (조회/생성/수정/삭제)

회사관리 (슈퍼관리자만)

회사 추가/수정/삭제는 슈퍼관리자만 가능합니다.
  • 회사 추가: 회사명, 사업자번호, 주소, 전화번호
  • 회사별 사용자 수, 부서 수 확인
  • 회사 정보 수정 및 삭제

11. 시스템 설정

양식 설정 (설정 > 양식 설정)

  • 인건비 등급(단가표) 관리: 등급명, 월단가, 일단가
  • 경비 항목 관리: 항목명, 계산방식(수동/비율/고정), 기본비율

문서 템플릿 (설정 > 문서 템플릿)

  • 견적서/계약서 템플릿 업로드 (DOCX, XLSX, PDF)
  • 부서별 또는 회사 전체 공유 설정

백업관리 (설정 > 백업관리)

  • 수동 백업: 데이터를 파일로 내보내기
  • 복원: 백업 파일에서 데이터 복원
